# 🪟 Wizdle.Wpf

A Windows desktop application for solving Wordle using the Wizdle engine.

Installers (self-contained, Windows only) are available on the [Wizdle releases page](https://github.com/lyndychivs/Wizdle/releases) (`wizdle-setup-x86.exe` or `wizdle-setup-x64.exe`).
